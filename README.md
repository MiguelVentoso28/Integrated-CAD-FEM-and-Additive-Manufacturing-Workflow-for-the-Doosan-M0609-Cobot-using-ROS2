# Doosan M0609 ROS 2 Automation Project

This repository contains the work developed for my Bachelor’s Degree Final Project in Mechanical Engineering.

The project focuses on the development of an integrated workflow for a collaborative robotic application using the **Doosan M0609 cobot**. The main goal is to combine **mechanical design**, **structural validation**, **additive manufacturing**, and **robotic control** in a ROS 2 environment.

## Project Description

The system is designed to perform an automated assembly task involving different components such as a screw, washers, and a cylindrical part.

The complete workflow includes:

* CAD design of a custom gripper
* FEM structural analysis
* Topology optimization
* 3D printing of prototypes
* ROS 2 integration
* MoveIt trajectory planning
* Robot simulation and validation

## Technologies Used

* ROS 2
* MoveIt
* RViz
* URDF / Xacro
* SolidWorks
* SolidWorks Simulation
* MATLAB
* Additive Manufacturing
* Computer Vision / YOLO

## Main Objective

The objective of this project is to develop a flexible robotic workflow capable of detecting, grasping, moving, and assembling components using a collaborative robot.
